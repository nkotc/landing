Notre idée

Une plateforme de cours particuliers dédiée à l’apprentissage du code informatique et à la sensibilisation aux problématiques du Numérique. Une mise en lien entre des coachs expérimentés et vos enfants! Des cours particuliers avec une pédagogie active grâce à laquelle ils pourront apprendre à apprendre et développer des applications qui leur seront vraiment utiles, pour susciter leur intérêt!
keyboard

Du code utile

On les appelle les Digital Natives mais il leur est souvent difficile de faire le lien entre leurs outils numériques du quotidien et la manière dont ils fonctionnent. L’idée est d’ancrer l’apprentissage de concepts algorithmiques et de la programmation dans la réalité! Ainsi les apprenants pourront coder des applications web ou mobiles dont ils peuvent avoir besoin.

directions_run

Des coachs experimentés

Le code est entré dans les programmes scolaires, mais les enseignants sont bien souvent un peu désarmés face à cette activité du programme pédagogique, souvent peu formés en amont. Les coachs NKOTC sont des personnes qui travaillent au quotidien dans la création de logiciels : développeurs, enseignants spécialisés dans la "vraie" vie, le code n'a plus de secret pour eux!

Toutes les clés du Numérique

Le code n’est pas le seul objectif d'apprentissage! Les questions autour de la sécurité, de la cryptologie, de la blockchain, de l'identité numérique, de l'éthique et de l'intelligence artificielle sont également abordées dans le programme. Notre objectif est de bien donner aux plus jeunes toutes les clés de compréhension du numérique!
